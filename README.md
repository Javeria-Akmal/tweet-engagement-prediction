# Elon Musk Tweet Engagement Prediction

Machine Learning project to classify tweet engagement (High/Low) based on historical tweet data.

##  Overview
This project uses a dataset of 12,206 tweets by Elon Musk (@elonmusk) from 2010 to 2022, including engagement metrics like Likes, Retweets, and Comments, to predict whether a tweet will have High or Low engagement.

##  Tech Stack
Python, XGBoost, Scikit-learn, Pandas, NumPy

##  What I Did
- Preprocessed and cleaned tweet engagement data
- Engineered features from likes, retweets, and comments
- Built a classification model to predict engagement level (High/Low)
- Compared model performance and tuned for best results

##  Results
Best performing model: **XGBoost**
- Accuracy: **89.43%**

##  How to Run
1. Clone this repo
2. Open the notebook in Jupyter or Google Colab
3. Install required libraries: `pip install pandas numpy scikit-learn xgboost`
4. Run all cells sequentially

##  Dataset
12,206 tweets by Elon Musk (2010–2022) with engagement metrics
